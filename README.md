## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### Navigate to project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```
